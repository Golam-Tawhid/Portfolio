# G.T. Fahad — Portfolio

Premium dark portfolio built with Next.js, TypeScript, Tailwind CSS, shadcn/ui, and Framer Motion.

## Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS** + shadcn/ui
- **Framer Motion**

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Structure

- `src/app/` — Next.js routes and global styles
- `src/components/` — UI sections, layout, motion, effects
- `src/lib/data/` — Centralized content (profile, projects, blog, etc.)
- `public/` — Static assets
